You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /github?query=users/amehlhase316/repos -- returns the name of the repos, id and login of owner
  • /multiply?num1=3&num2=4 -- (options: 1-100) returns the result
  • /combo?color1=2&color2=3 -- (options: red=1, yellow=2, blue=3) returns the color combination outcome
  • /guessNumbers?num1=5&num2=17 -- (options: 0-20) compares input numbers to a randomly selected number
